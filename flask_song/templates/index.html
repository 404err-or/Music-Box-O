노래 업로드 방법 Link

1. "오디오 파일 업로드" 버튼을 클릭하여 "{가수이름} - {노래제목}.mp3" 형식의 파일을 선택한다.
    만약 형식이 이와 같지 않다면 파일명을 수정하여 업로드한다.
2. 해당 파일들이 웹페이지에 정상적으로 뜨면, 이제 mp3 파일을 업로드할 준비가 되었다.
3. category란을 선택하여 해당 노래가 어떤 카테고리에 들어가는지 최소 1개 선택한다.
4. Upload 버튼을 클릭하면 적합성을 판단한 후, 업로드한다.
5. 해당 곡을 업로드하고 싶지 않다면 그 옆에 있는 X 버튼을 누른다.

노래 삭제 방법 Link

1. 상단에서 카테고리를 선택한다.
2. 해당 카테고리에 있는 노래들이 뜨면, 해당 카테고리에서 지우고 싶은 노래의 X 버튼을 누른다.
3. 정말로 삭제할 것인지 묻는 창에서 취소를 누르면 취소되고, 확인을 누르면 삭제가 된다.
4. 만약 all 카테고리에서 삭제 시, 모든 카테고리에서 삭제된다.
5. 만약 카테고리에서 삭제했는데, 이후 그 곡이 어떤 카테고리에도 속해있지 않으면, all 카테고리에서도 삭제된다.0

노래 재생 방법 Link

1. 재생 버튼을 눌러 노래를 재생할 수도 있고, 좌측(또는 상단)의 playlist에서 노래를 선택하여 노래를 재생할 수 있다.
2. 중지 버튼을 눌러 노래를 중지할 수 있다. 중지 이후 다시 시작 시, 중단 시점에서 노래가 다시 재생된다.

곡 변경 재생 방법 Link

1. 다음 곡 버튼을 누르거나, 이전 곡 버튼을 누르거나, playlist에서 다음곡을 눌러 직접 다음곡으로 넘길 수 있다.
2. 만약 노래가 끝나면 자동으로 곡이 넘어간다. 3. 이전 버튼을 눌렀을 시, 노래가 시작한지 5초 이내이면 이전곡을 재생하고, 그렇지 않으면 해당 곡을 처음부터 재생한다.

Playlist 변경 방법 Link

1. 좌측(또는 상단)의 카테고리를 선택하여 카테고리별 재생을 할 수 있다.
2. 노래가 랜덤하게 재생되기를 원한다면, 우측(또는 하단)의 셔플 버튼을 클릭하면 랜덤으로 보여진다. (중요)3. 현재 playlist가 마음에 든다면 save 버튼을 꼭 눌러야 노래 재생에 적용된다. 4. 노래가 재생 중일때 playlist가 변경되었을 시, 재생중인 노래가 playlist 안에 들어있다면 그냥 진행한다. 5. 반면에 없다면, 노래 재생이 멈춘다. 그 후 노래를 재생해주어야한다.

볼륨 조절 방법 Link

우측(또는 하단)의 vol 버튼을 누르면 dropup 창에서 scroll 바 창이 뜨는데 해당 스크롤을 위치시키면 노래 재생이 된다.
